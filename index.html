24周系统学习 4大进阶阶段 企业级项目

学习提示

每天至少保证 2小时编码时间 ，周末可安排综合练习

遇到卡点：善用 AI 、 Stack Overflow 、 MDN文档

扩展学习：关注 Web Vitals性能指标 、 WebAssembly 前沿技术

按照此路线，24周后可达到 初中级前端工程师 水平，具备企业项目开发能力。记得在每个阶段结束后更新简历，记录技术成长！

详细学习路线

1

阶段一：基础入门（4周）

掌握HTML/CSS/JavaScript基础，搭建开发环境

Week 1：HTML & CSS基础

  • HTML5语义化标签：header/nav/main/article/section/footer
  • CSS盒模型：margin/padding/border/content-box vs border-box
  • Flexbox布局：flex-direction/justify-content/align-items/flex-wrap
  • Grid布局基础：grid-template-columns/rows、grid-gap
  • CSS选择器：类选择器、ID选择器、属性选择器、伪类选择器
  • 响应式设计：媒体查询、移动优先原则、rem/em单位
  • CSS变量：自定义属性、主题切换基础
实战：简单个人简历 实战：优化版个人简历 实战：CSS 布局样式 实战：商城首页

Week 2：JavaScript基础

  • 变量声明：var/let/const区别、变量提升
  • 数据类型：原始类型（string/number/boolean/null/undefined/symbol）
  • 引用类型：Object/Array/Function/Date/RegExp
  • 运算符：算术/比较/逻辑/赋值/三元运算符
  • 流程控制：if/else、switch/case、for/while/do-while循环
  • 函数基础：函数声明/表达式、参数传递、返回值
  • 作用域：全局作用域、函数作用域、块级作用域
  • 闭包原理：词法作用域、内存管理
实战：计算器小工具

Week 3：DOM操作 & 事件

  • DOM查询：getElementById/querySelector/querySelectorAll
  • DOM操作：createElement/appendChild/removeChild
  • 属性操作：setAttribute/getAttribute/classList
  • 样式操作：style属性、className、classList API
  • 事件监听：addEventListener/removeEventListener
  • 事件对象：event.target/event.preventDefault()
  • 事件流：捕获阶段、目标阶段、冒泡阶段
  • 事件委托：利用冒泡机制优化性能
实战：TodoList待办事项应用

Week 4：Git入门 & 网络基础

  • Git基础命令：git init/add/commit/push/pull/clone
  • Git配置：user.name/user.email、SSH密钥配置
  • GitHub操作：创建仓库、Fork、Pull Request流程
  • HTTP协议：请求方法（GET/POST/PUT/DELETE）
  • 状态码：2xx成功、3xx重定向、4xx客户端错误、5xx服务器错误
  • 请求头/响应头：Content-Type/Accept/User-Agent
  • 浏览器缓存：强缓存、协商缓存机制
  • 跨域问题：CORS、JSONP原理
实战：将TodoList代码提交至GitHub
2

阶段二：核心技能提升（8周）

掌握ES6+、框架基础、工程化工具

Week 5-6：ES6+ & 异步编程

  • ES6+语法：let/const、模板字符串、解构赋值、展开运算符
  • 箭头函数：this绑定、简写语法、适用场景
  • 模块化：import/export、CommonJS vs ES Modules
  • 异步编程：回调函数、Promise对象、async/await
  • Promise进阶：Promise.all/Promise.race/Promise.allSettled
  • Fetch API：GET/POST请求、请求头配置、错误处理
  • 错误处理：try/catch、Error对象、自定义错误类型
  • 迭代器与生成器：Symbol.iterator、yield关键字
实战：天气查询应用（调用开放API）

Week 7-8：前端框架（React/Vue二选一）

  • React路线：JSX语法、函数组件/类组件、Props/State
  • React核心概念：虚拟DOM、Diff算法、组件生命周期
  • React事件处理：合成事件、事件绑定、事件传参
  • React条件渲染：&&运算符、三元运算符、map()渲染列表
  • Vue路线：模板语法、选项式API、响应式原理
  • Vue指令系统：v-if/v-show/v-for/v-bind/v-on
  • Vue计算属性：computed vs methods、侦听器watch
  • Vue组件通信：Props/$emit、provide/inject
实战：电影列表页（渲染API数据）

Week 9：组件化开发

  • React Hooks：useState/useEffect/useContext/useReducer
  • 自定义Hooks：逻辑复用、副作用封装
  • 组件生命周期：挂载/更新/卸载阶段、useEffect依赖数组
  • Vue Composition API：ref/reactive/computed/watch
  • Vue 3特性：setup函数、Teleport、Suspense
  • 组件设计原则：单一职责、高内聚低耦合
  • 组件通信模式：父子通信、兄弟通信、跨级通信
  • 组件复用策略：高阶组件、Render Props、插槽
实战：评论区组件（支持增删评论）

Week 10：路由管理

  • React Router：BrowserRouter/Routes/Route/Link组件
  • 路由参数：useParams/useSearchParams/useLocation
  • 编程式导航：useNavigate/useHistory
  • 路由守卫：路由拦截、权限验证
  • Vue Router：createRouter/router-view/router-link
  • 动态路由：路由参数、查询参数、路径匹配
  • 嵌套路由：子路由配置、命名视图
  • 路由懒加载：组件懒加载、代码分割优化
实战：多页面博客系统

Week 11：工程化工具链

  • Webpack核心概念：Entry/Output/Loader/Plugin/Mode
  • 常用Loader：babel-loader/css-loader/file-loader
  • 常用Plugin：HtmlWebpackPlugin/CleanWebpackPlugin
  • 开发服务器：webpack-dev-server、热更新HMR
  • Babel配置：presets/plugins、polyfill配置
  • 代码分割：动态import()、SplitChunksPlugin
  • 打包优化：Tree Shaking、压缩优化、缓存策略
  • 环境配置：开发/生产环境差异化配置
实战：手动搭建项目脚手架

Week 12：网络进阶 & Git协作

  • HTTPS/TLS：加密原理、证书验证、中间人攻击防护
  • Cookie/Session：存储机制、安全性、SameSite属性
  • Token认证：JWT结构、签名验证、刷新机制
  • RESTful API：资源设计、HTTP方法语义、状态码规范
  • GraphQL基础：查询语言、类型系统、Resolver
  • Git分支策略：Git Flow、Feature Branch、Release Branch
  • Git高级操作：rebase/squash/cherry-pick/stash
  • 团队协作流程：Pull Request、Code Review、冲突解决
实战：团队协作修复GitHub Issue
3

阶段三：企业级技术栈（8周）

掌握状态管理、TypeScript、测试

Week 13-14：状态管理库

  • Redux核心概念：Store/Action/Reducer、单向数据流
  • React-Redux：Provider/connect/useSelector/useDispatch
  • Redux Toolkit：createSlice/configureStore/createAsyncThunk
  • 中间件机制：redux-thunk/redux-saga/redux-observable
  • Pinia状态管理：defineStore/storeToRefs/pinia插件
  • 状态持久化：localStorage/sessionStorage集成
  • 状态调试：Redux DevTools、时间旅行调试
  • 性能优化：reselect记忆化、不可变数据更新
实战：购物车系统（全局状态管理）

Week 15-16：TypeScript

  • 基础类型：string/number/boolean/array/tuple/enum/any
  • 类型注解：变量类型、函数参数/返回值类型
  • 接口与类型别名：interface vs type、可选属性、只读属性
  • 泛型编程：泛型函数、泛型接口、泛型约束
  • 高级类型：联合类型、交叉类型、类型守卫、映射类型
  • React + TypeScript：组件Props类型、Hooks类型、事件类型
  • Vue + TypeScript：组合式API类型、Props类型、Emit类型
  • 工具类型：Partial/Required/Readonly/Pick/Omit
实战：重构TodoList为TS版本

Week 17：测试开发

  • Jest基础：describe/it/expect、匹配器、异步测试
  • 测试覆盖率：语句/分支/函数/行覆盖率分析
  • React Testing Library：render/fireEvent/screen查询方法
  • 组件测试：Props测试、事件测试、状态测试
  • Vue Test Utils：mount/shallowMount/wrapper方法
  • Mock与Stub：jest.fn/jest.mock/模块模拟
  • 集成测试：用户行为模拟、端到端测试基础
  • 测试最佳实践：测试金字塔、测试驱动开发TDD
实战：为组件编写测试用例

Week 18：性能优化

  • 懒加载技术：React.lazy/Suspense、Vue异步组件
  • 代码分割：路由级分割、组件级分割、动态import
  • 防抖与节流：lodash.debounce/throttle、自定义实现
  • 虚拟滚动：react-window/vue-virtual-scroller原理
  • Chrome DevTools：Performance面板、Memory面板、Lighthouse
  • React性能优化：React.memo/useMemo/useCallback
  • Vue性能优化：v-once/v-memo、计算属性缓存
  • 网络优化：HTTP/2、CDN、资源压缩、缓存策略
实战：优化电影列表页加载速度

Week 19-20：进阶网络与安全

  • WebSocket协议：握手过程、消息帧格式、心跳机制
  • Socket.io库：事件驱动、房间管理、广播消息
  • XSS攻击防护：输入验证、输出编码、CSP策略
  • CSRF攻击防护：Token验证、SameSite Cookie、Referer检查
  • JWT认证：Header/Payload/Signature结构、签名验证
  • OAuth 2.0：授权码模式、隐式模式、客户端凭证模式
  • HTTPS安全：TLS握手、证书验证、HSTS策略
  • 安全最佳实践：密码哈希、盐值加密、安全头设置
实战：实时聊天室（WebSocket + JWT）
4

阶段四：项目实战（4周）

完成企业级全栈项目

Week 21-22：全栈项目搭建

  • 项目架构设计：前后端分离架构、API设计规范、数据流规划
  • 前端技术栈：React 18 + TypeScript 或 Vue 3 + Composition API
  • 后端技术栈：Node.js + Express 或 Nest.js框架选择
  • 数据库设计：MongoDB文档模型、Firebase实时数据库、数据关系设计
  • 用户认证系统：JWT实现、密码加密、权限角色管理
  • API接口开发：RESTful设计、错误处理、数据验证中间件
  • 前端路由配置：React Router v6 或 Vue Router 4配置
  • 状态管理集成：Redux Toolkit + RTK Query 或 Pinia状态管理

项目要求：

  • 使用Git Flow管理代码
  • 编写单元测试覆盖核心逻辑
  • 实现CI/CD自动部署（Vercel/Netlify）

Week 23-24：项目优化与部署

  • 性能分析工具：Webpack Bundle Analyzer、Lighthouse、Chrome DevTools
  • 代码分割优化：路由级懒加载、组件级代码分割、Tree Shaking
  • 缓存策略优化：浏览器缓存、CDN加速、Service Worker缓存
  • SEO优化技术：Next.js SSR、Nuxt.js SSG、Meta标签优化
  • 容器化部署：Dockerfile编写、镜像构建、容器编排基础
  • CI/CD流水线：GitHub Actions、自动化测试、自动部署流程
  • 监控与日志：错误监控、性能监控、用户行为分析
  • 安全加固：环境变量管理、安全扫描、依赖漏洞检查

交付物：

  • GitHub仓库（含完整Commit历史）
  • 线上演示地址
  • 项目文档（README.md说明架构、部署步骤）

关键贯穿点

每周Git实践

  • 第1-4周：个人仓库每日提交
  • 第5周起：使用Feature Branch工作流
  • 项目阶段：模拟团队PR（Pull Request）审核

网络学习节奏

  • 每周学习1个HTTP核心概念（如缓存、CORS）
  • 项目阶段结合Chrome Network面板分析请求

软技能培养

  • 第8周：参与GitHub开源项目（提交文档修复）
  • 第16周：撰写技术博客（如"TypeScript类型体操心得"）
