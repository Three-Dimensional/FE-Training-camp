零基础 24 周前端学习路线图

从入门到企业级项目实战，系统掌握现代前端开发技能

24周系统学习 4大进阶阶段 企业级项目

学习提示

每天至少保证 2小时编码时间 ，周末可安排综合练习

遇到卡点：善用 AI 、 Stack Overflow 、 MDN文档

扩展学习：关注 Web Vitals性能指标 、 WebAssembly 前沿技术

按照此路线，24周后可达到 初中级前端工程师 水平，具备企业项目开发能力。记得在每个阶段结束后更新简历，记录技术成长！

详细学习路线

1

阶段一：基础入门（4周）

掌握HTML/CSS/JavaScript基础，搭建开发环境

Week 1：HTML & CSS基础

  • HTML5标签、语义化
  • CSS选择器、盒模型、Flex布局
  • 响应式设计（Media Query）
实战：简单个人简历 实战：优化版个人简历 实战：CSS 布局样式 实战：商城首页

Week 2：JavaScript基础

  • 变量、数据类型、运算符
  • 条件语句、循环
  • 函数、作用域、闭包
实战：计算器小工具

Week 3：DOM操作 & 事件

  • DOM元素增删改查
  • 事件监听、事件冒泡/捕获
实战：TodoList待办事项应用

Week 4：Git入门 & 网络基础

  • Git安装、 git init/add/commit/push
  • GitHub创建仓库、提交代码
  • HTTP协议（GET/POST）、状态码（200/404）
实战：将TodoList代码提交至GitHub
2

阶段二：核心技能提升（8周）

掌握ES6+、框架基础、工程化工具

Week 5-6：ES6+ & 异步编程

  • let/const、模板字符串、解构赋值
  • Promise、async/await
  • Fetch API调用数据
实战：天气查询应用（调用开放API）

Week 7-8：前端框架（React/Vue二选一）

  • React路线 ：JSX语法、组件、Props/State
  • Vue路线 ：模板语法、响应式、指令
实战：电影列表页（渲染API数据）

Week 9：组件化开发

  • 组件生命周期
  • 状态管理（React Hooks / Vue Composition API）
实战：评论区组件（支持增删评论）

Week 10：路由管理

  • React Router / Vue Router
  • 动态路由、嵌套路由
实战：多页面博客系统

Week 11：工程化工具链

  • Webpack基础配置（Loader/Plugin）
  • Babel转译ES6+
实战：手动搭建项目脚手架

Week 12：网络进阶 & Git协作

  • HTTPS/TLS、Cookie/Session
  • RESTful API设计规范
  • Git分支管理（ git branch/merge/rebase ）
实战：团队协作修复GitHub Issue
3

阶段三：企业级技术栈（8周）

掌握状态管理、TypeScript、测试

Week 13-14：状态管理库

  • Redux（React） / Pinia（Vue）
  • 异步Action、中间件
实战：购物车系统（全局状态管理）

Week 15-16：TypeScript

  • 类型注解、接口、泛型
  • TS与React/Vue集成
实战：重构TodoList为TS版本

Week 17：测试开发

  • Jest单元测试
  • React Testing Library / Vue Test Utils
实战：为组件编写测试用例

Week 18：性能优化

  • 懒加载、代码分割
  • 防抖/节流、虚拟滚动
  • Chrome DevTools性能分析
实战：优化电影列表页加载速度

Week 19-20：进阶网络与安全

  • WebSocket实时通信
  • XSS/CSRF防御策略
  • JWT身份验证
实战：实时聊天室（WebSocket + JWT）
4

阶段四：项目实战（4周）

完成企业级全栈项目

Week 21-22：全栈项目搭建

  • 技术选型 ：React + Node.js 或 Vue + Express
  • 数据库 ：MongoDB/Firebase
  • 功能模块 ：用户系统、数据看板、权限控制

项目要求：

  • 使用Git Flow管理代码
  • 编写单元测试覆盖核心逻辑
  • 实现CI/CD自动部署（Vercel/Netlify）

Week 23-24：项目优化与部署

  • 性能优化（Bundle分析、CDN）
  • SEO优化（SSR方案：Next.js/Nuxt.js）
  • 容器化部署（Docker基础）

交付物：

  • GitHub仓库（含完整Commit历史）
  • 线上演示地址
  • 项目文档（README.md说明架构、部署步骤）

关键贯穿点

每周Git实践

  • 第1-4周：个人仓库每日提交
  • 第5周起：使用Feature Branch工作流
  • 项目阶段：模拟团队PR（Pull Request）审核

网络学习节奏

  • 每周学习1个HTTP核心概念（如缓存、CORS）
  • 项目阶段结合Chrome Network面板分析请求

软技能培养

  • 第8周：参与GitHub开源项目（提交文档修复）
  • 第16周：撰写技术博客（如"TypeScript类型体操心得"）
